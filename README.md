# 🐾 Cabinet Vétérinaire — Dr. Boulemaiz Azzeddine

Logiciel de gestion pour cabinet vétérinaire, développé avec Flask et SQLite.

## ✨ Fonctionnalités

- 📦 Gestion du stock de médicaments
- 🗂️ Classification complète des médicaments (20 catégories)
- 📥 Entrée en stock par lots
- 💊 Point de vente
- 📊 Rapports et bénéfices
- 🌐 Interface trilingue (FR / AR / EN)

## 🛠️ Technologies

- Python / Flask
- SQLite
- HTML / CSS / JavaScript

## 🚀 Installation

```bash
pip install -r requirements.txt
python app.py
```
